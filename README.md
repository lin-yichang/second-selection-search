# Second Context Search Extension

A Firefox browser extension that adds a second search engine to the context menu.

It is useful, for example, if you want a traditional search engine (e.g., Google) for general searches, and an online dictionary or Wikipedia as a second search engine.

## Features

- Supports your browser's search engines and custom search URLs
- Has a keyboard shortcut for quicker access

## Installation

### Install from Firefox Add-ons

You can install the extension directly from Firefox Add-ons:

https://addons.mozilla.org/addon/second-context-search

Whether you enjoy the extension or have suggestions, please consider rating it and leaving a comment there.

### Install locally for development

1. Download or clone this repository.
2. Open Firefox and navigate to `about:debugging`.
3. Click **This Firefox** in the sidebar.
4. Click **Load Temporary Add-on**.
5. Select the `manifest.json` file from this project.

The extension will be installed temporarily for the current Firefox session.

## Usage

1. Select the text on a web page that you want to search for.
2. Right-click to open the context menu.
3. Click **2nd Search Engine**, or hit the keyboard shortcut <kbd>2</kbd>.

The first time you use the extension, the options page opens automatically, where you can easily set up a second search engine. The options page can also be opened from the browser's add-ons manager.
