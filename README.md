# Second Context Search Extension

A Firefox browser extension that adds a second search engine to the context menu.

## Features

- Choose between installed search engines or define a custom independent search URL
- Easy configuration through the extension options page

## Installation

1. Download or clone this repository
2. Open Firefox and navigate to `about:debugging`
3. Click "This Firefox" in the sidebar
4. Click "Load Temporary Add-on"
5. Select the `manifest.json` file from this project

The extension will be temporarily installed.

## Usage

1. Select the text on a web page that you want to search for
2. Right-click to open the context menu
3. Click **2nd Search Engine**

The first time you use the extension, the options page opens automatically, where you can set up a second search engine. The options page can also be opened from the browser's add-ons manager.
